# modelnet40_ply_hdf5_2048
## About
- Tools for ModelNet40 dataset.
- The following models use this dataset.
  - [PointNet](../../PointNet/README.md)
  - [DGCNN](../../DGCNN/README.md)

## Download
- Download dataset.
- ```bash
  python download.py -p /path/to/dataset
  ```
  - args
    - `-p`: save location
